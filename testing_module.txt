## Tests `NLTK` token annotator function:
These modules are designed to take a string as input and output a tuple annotated with hex colour tags.

```python

>>> sent = "This is a test."
>>> nltk_token_annotator(text = sent, model = "whitespace")
[('This', '', '#8ef'), ('is', '', '#faa'), ('a', '', '#afa'), ('test.', '', '#fea')]

>>> nltk_token_annotator(text = sent, model = "punkt")
[('This is a test.', '', '#8ef')]

```

## Testing `SpaCy` annotator function:

```python

>>> nlp = spacy.load("en_core_web_sm")
>>> doc = nlp(sent)

>>> spacy_token_annotator(doc = doc, model = "sent")
[('This is a test.', '', '#8ef')]
```
